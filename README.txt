# SGBD Simplificado

Sistema de Gerenciamento de Banco de Dados simplificado baseado no modelo relacional.

## O que foi feito?

1. Criar uma tabela
2. Listar todas as tabelas
3. Criar uma nova linha na tabela
4. Listar todos os dados de uma tabela
6. Apagar valor de uma tabela
7. Apagar uma tabela


## O que não foi feito?

5. Pesquisar valor em uma tabela


## O que faria de forma diferente?

Utilizaria mais boas práticas de programação.


## Como compilar o projeto?

1. Compile os arquivos.
2. Execute o programa.

```console
make
make run
```
